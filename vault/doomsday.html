How to Use the Doomsday Method

Step 1: Learn the Anchor Days

Each month has a memorable anchor date (like 4/4, 6/6, 8/8, 10/10). These always fall on the same weekday as the last day of February.

Step 2: Find the Century Code

Each century corresponds to a weekday code. For example: 1800s = Friday, 1900s = Wednesday, 2000s = Tuesday.

Step 3: Calculate the Year Code

Use the last two digits of the year. Divide by 12, add the remainder, add the number of 4s in the remainder, then combine everything with the century code to get the weekday.

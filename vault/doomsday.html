Doomsday Method – Classic Calculation

Let’s learn how to calculate any day of the week using mental math!

Example: We will calculate the day of the week for 19.05.1985 (19th May 1985)

Step 1: Get the Last Two Digits of the Year 📅

1985 → 85

Step 2: Divide by 4 ➗

85 ÷ 4 = 21 (keep only the integer)

Step 3: Add Year + Quotient ➕

85 + 21 = 106

Step 4: Add the Century Code 🧭

For the 1900s, the code is 0 → Total = 106

Step 5: Add the Month Code 🗓️

May = 1 → 106 + 1 = 107

Step 6: Add the Day 📆

Day = 19 → 107 + 19 = 126

Step 7: Divide by 7 and Find the Remainder 7️⃣

126 ÷ 7 = 18 remainder 0

The remainder tells you the weekday!

0 = Sunday → So 19.05.1985 was a Sunday

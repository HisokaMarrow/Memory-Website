Leap-Adjusted Doomsday Method

This version of the Doomsday rule uses a simplified modular approach with leap year correction. Follow the steps below to determine the day of the week for any date.

Step 1: Add Base Values

Add together:

  • The day of the month
  • The month code from the table below
  • The century code from the table below

Step 2: Add Year Values

From the last two digits of the year:

  • Add the year itself
  • Add the integer part of the year divided by 4 (i.e., Math.floor(year / 4))

Step 3: Modulo 7

Add all values from Step 1 and Step 2, then take the result modulo 7. This gives the weekday code:

RemainderDay
0Sunday
1Monday
2Tuesday
3Wednesday
4Thursday
5Friday
6Saturday

Step 4: Leap Year Adjustment

If the date is in January or February and the year is a leap year, subtract 1 from your result.

Leap Year Rule: A year is a leap year if it is divisible by 4 — except century years, which must be divisible by 400.

If your adjusted result is -1, treat it as 6 (wraparound).

Example

Date: February 10, 1996

  • Day = 10
  • Month Code = 3
  • Century Code = 0 (1900s)
  • Year = 96
  • 96 ÷ 4 = 24
  • Sum = 10 + 3 + 0 + 96 + 24 = 133
  • 133 mod 7 = 0 → Sunday
  • Leap Year + February = Subtract 1 → Saturday

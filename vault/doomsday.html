Doomsday Method – Classic Calculation

A mental algorithm to find any day of the week — no calendar needed!

Step 1: Get the Last Two Digits of the Year 📅

Take the last two digits of the year.
Example: 1995 → 95

Step 2: Divide by 4 ➗

Divide 95 by 4 and keep the whole number only: 95 ÷ 4 = 23

Step 3: Add Year + Quotient ➕

Add both numbers together: 95 + 23 = 118

Step 4: Add the Century Code 🧭

Use the century code based on the century of the year: 1900s = 0 → 118 + 0 = 118

Step 5: Add the Month Code 🗓️

Look up the month code (see table below).
Example: May = 1 → 118 + 1 = 119

Step 6: Add the Day 📆

For May 10: 119 + 10 = 129

Step 7: Divide by 7 and Find the Remainder 7️⃣

129 ÷ 7 = 18 R 3
The remainder tells you the weekday!
